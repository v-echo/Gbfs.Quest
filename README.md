# Gbfs.Quest

A small quiz-type game, built using CLI visual components, with questions based off of GBFS feeds.

## Prerequisites
- Visual Studio 2026
- .NET 10 SDK

## Usage
- Build and run

## Features
- Authentication (in-memory)
- Random generation of questions for every run
- Bridging version differences through Json converters
- Async and parallel workflow dispatching updates to the UI thread
- Native AoT enabled: can run on any OS/arch if you build for it
- No external dependencies (like services, integrations or databases)

Note: The UI is configured for mouse interaction, rather than the classic command-based approach.
